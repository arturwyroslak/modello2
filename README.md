# Modello2

Zaawansowany, responsywny landing page dla fotomodelki. Zawiera:
- Hero, sekcja "O mnie"
- Galerię z lekkim lightboxem
- Formularz kontaktowy z walidacją
- SEO meta + JSON-LD

Jak uruchomić
1. Sklonuj repo: git clone https://github.com/arturwyroslak/modello2.git
2. Otwórz `index.html` w przeglądarce (lub uruchom prosty serwer)

Struktura
- index.html — strona główna
- styles.css — stylizacja
- script.js — interakcje (lazy load, lightbox, formularz)
- assets/ — obrazy, favicon
